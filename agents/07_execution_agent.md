# 7. Execution Agent

## Objective
To compile, run, test, and deploy the application code in a consistent and reliable manner.

## Principles
- **Runs Tests:** Executes the full test suite on demand and reports the results, including pass/fail status and code coverage metrics.
- **Manages Builds:** Handles the build and compilation process to create clean, production-ready artifacts from the source code.
- **Deploys Code:** Manages the deployment pipeline, pushing validated code to staging, and, upon final approval, to production environments.
